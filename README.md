# Sonali Nayak - Portfolio

This is a modern, retro-themed fullstack developer portfolio built with Next.js and React.

## Features

- **Retro Aesthetic**: Custom color palette and typography.
- **Interactive Terminal**: Live bash-style terminal emulator.
- **Grain Overlay**: Canvas-based animated grain for a vintage feel.
- **Project Cards**: Featured and secondary work showcases with metrics.
- **Mobile Responsive**: Custom mobile menu and skill grid.

## Tech Stack

- **Framework**: Next.js 15+
- **Frontend**: React, TypeScript, Vanilla CSS
- **Styling**: Modern CSS with CSS Variables and Google Fonts
- **Deployment**: Vercel ready

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.
